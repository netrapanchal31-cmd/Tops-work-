🔹 38. Decorator

A decorator in Python is a function that adds extra functionality to another function without changing its original code.
Decorator = wrapper function that modifies another function

  example:
def my_decorator(func):
    def wrapper():
        print("Before function runs")
        func()
        print("After function runs")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

say_hello()
 Output:
Before function runs
Hello!
After function runs
Explanation (Simple)
my_decorator → decorator function
say_hello → original function
@my_decorator → applies decorator
wrapper → adds extra behavior before & after

 Key Point

Add extra features (logging, checking, timing, etc.)
Without editing the original function

🔹 39. Closures 
Definition:
 A closure is a function that remembers the values of variables from its outer function even after the outer function has finished executing.
Closure = inner function + remembers outer function data
example:
def outer_function(x):
    def inner_function(y):
        return x + y   # uses outer variable x
    return inner_function

add_five = outer_function(5)

print(add_five(3))
 Output:
8
 Explanation (Simple)
outer_function(5) runs and returns inner_function
Even after outer function ends, inner_function remembers x = 5
When we call add_five(3) → it does 5 + 3 = 8

 Key Points

Closure stores data from outer function
Useful for:
Data hiding
Creating customized functions
No need for global variables
